# Step 2: コマンド行の解析

シェルに入力されたコマンド行を解析し、コマンドと引数に分割する機能を実装します。

## 学ぶこと

- 文字列のトークン分割 (`strtok`)
- 配列とポインタの応用

## やってみましょう

1. [practice.c](./practice.c) の TODO に従ってコードを埋めてみましょう。

2. コンパイル・実行してみましょう。
```shell
cd step2_parse/
gcc practice.c -o practice
./practice
```

3. 動作確認：  
    - プロンプトが表示されたら、複数の単語からなるコマンドを入力してください。  
      例: `ls -la /usr/bin`  
    - コマンドと引数の数が正しく表示されることを確認します。
